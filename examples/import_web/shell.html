vasset Import Web Demo

Upload one or more files, or an entire folder, then run vasset-import inside the browser-backed filesystem. The demo writes your uploads into Emscripten MEMFS, imports them under /workspace/assets, and prints a summary of the imported resources.

Standalone textures and splat files are the easiest to test. For mesh assets with sibling textures, use the folder picker so relative paths are preserved.
